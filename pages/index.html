How It Works

Pages Store Links Each page is a list of links. Anyone can view or add to it.
Always Public There are no private pages—anyone with the name has full access.

No Directory Pages can only be found by guessing or knowing the name.
Obscurity = Privacy A common name (e.g., /groupchatvideos) is easy to guess. A random one (e.g., /b7Qx9fL2) is nearly impossible to find.
